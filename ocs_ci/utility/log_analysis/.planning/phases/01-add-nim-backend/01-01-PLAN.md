Create `ocs_ci/utility/log_analysis/ai/nim_backend.py` with a `NimBackend` class that subclasses `ClaudeCodeBackend`.

The class overrides only `_call_claude` and `_call_claude_agentic` to inject LiteLLM proxy env vars into the subprocess environment before delegating to `super()`.

Implementation details:

1. Module docstring: explain this routes Claude Code CLI through LiteLLM proxy to NVIDIA NIM models.

2. Define a module-level dict for the env vars to inject:
```python
NIM_ENV_VARS = {
    "ANTHROPIC_BASE_URL": "http://localhost:4000",
    "ANTHROPIC_API_KEY": "sk-litellm-local",
    "ANTHROPIC_MODEL": "claude-sonnet-4-6",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "claude-opus-4-6",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "claude-sonnet-4-6",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "claude-haiku-4-5",
}
```

3. `NimBackend` class:
   - Inherits from `ClaudeCodeBackend` (import from `ocs_ci.utility.log_analysis.ai.claude_code_backend`)
   - `__init__`: calls `super().__init__(model=model, max_budget_usd=max_budget_usd, save_prompts_dir=save_prompts_dir)` with same defaults
   - Override `_call_claude(self, prompt, json_schema, context="")`:
     - Save current env values for NIM_ENV_VARS keys: `saved = {k: os.environ.get(k) for k in NIM_ENV_VARS}`
     - Set `os.environ[k] = v` for each k,v in NIM_ENV_VARS
     - Call `result = super()._call_claude(prompt, json_schema, context)` in a try/finally
     - In finally block, restore saved values: if saved[k] is None, pop k from os.environ; else set os.environ[k] = saved[k]
     - Return result
   - Override `_call_claude_agentic(self, prompt, context="", allowed_tools="Bash")`:
     - Same pattern: save, set NIM_ENV_VARS, call super(), restore in finally
   - Override `is_available(self)` to also check that LiteLLM proxy is reachable:
     - Call `super().is_available()` first (checks claude CLI exists)
     - If that passes, try `requests.get("http://localhost:4000/health", timeout=2)` and return `response.ok`
     - On any exception (ConnectionError, Timeout), log a warning and return False
     - Import `requests` at module level (already a project dependency)

4. Add module-level `logger = logging.getLogger(__name__)`
